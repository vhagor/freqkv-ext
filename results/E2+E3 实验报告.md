# E2+E3 实验报告：离线率失真 + 预算分配（主张 C2）

## 1. 实验目的

验证主张 **C2**：RST-KV 的"带通主体 + 稀疏残差 + 率失真分配"在相同压缩预算下，
重建/注意力输出误差是否严格低于 FreqKV 的均匀 DCT 低通。

这是**判生死**的实验——完全不训练，在真实抓取的 K/V 上直接度量四种编码的重建质量，
决定是否值得投入 H100 做大规模训练（E4）。

## 2. 实验配置

| 参数 | 值 |
|------|-----|
| 模型 | LLaMA-2-7B (`/root/llama2-7b/`) |
| RoPE base | 10000.0 |
| 序列长度 N | 2048 tokens |
| 校准样本数 | 8 |
| 校准数据 | PG-19 test split（预分词） |
| 分析层 | 0, 8, 16, 31 |
| 压缩比 γ | 0.5, 0.25, 0.125, 0.0625, 0.03125 |
| RST 默认 α | 0.7 |
| α 扫描值 | 0.0, 0.5, 0.7, 0.85, 1.0 |
| K 域设置 | natural（DCT/wavelet 在 pre-RoPE 上压缩，DFT-RoPE/RST 在 post-RoPE 上压缩）|
| 残差域 | time（时域 top-k）|
| 精度 | float16 |
| head_dim | 128 |

### 四种编码

| 编码 | 说明 |
|------|------|
| **DCT** | FreqKV 基线：DCT-II 低通，保留最低 γN 个系数 |
| **DFT-RoPE** | per-pair θ 中心带通，保留 γN 个 bin |
| **Wavelet** | DWT(db4) 硬阈值，保留最大的 γN 个系数 |
| **RST** | 混合码：DFT-RoPE bulk（α·γ 预算）+ 时域稀疏残差（(1-α)·γ 预算）|

### 三种误差度量

| 度量 | 公式 | 意义 |
|------|------|------|
| K relerr | \|\|K − K̂\|\|_F / \|\|K\|\|_F | Key 重建精度 |
| V relerr | \|\|V − V̂\|\|_F / \|\|V\|\|_F | Value 重建精度 |
| **attn relerr** | \|\|softmax(qK^T)V − softmax(qK̂^T)V̂\|\| / \|\|softmax(qK^T)V\|\| | **注意力输出精度**（最终影响模型的核心度量）|

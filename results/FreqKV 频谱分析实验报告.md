# FreqKV 频谱分析实验报告

## 1. 实验概述

### 1.1 目的

本实验旨在验证 freqkv-ext 项目提出的两个核心 DSP 假设：

1. **DFT-RoPE 调制假设**：RoPE 对 LLaMA 的 Key 状态施加的旋转，在频域上等价于对每对隐维 (pair i) 做 θ_i 的频移（DFT 调制定理）。如果成立，则 FreqKV 的均匀 DCT 低通滤波在低索引 pair（大 θ）上会丢失关键频段，RoPE 匹配的带通滤波可以改善压缩质量。

2. **小波稀疏性假设**：KV 状态沿序列轴存在时频局部结构（如 needle token、代码符号等局部事件），小波基比 DCT/DFT 全局基能以更少的系数捕获同等比例的能量，因此小波压缩可能优于 FreqKV 的 DCT。

### 1.2 实验配置

| 参数 | 值 |
|------|-----|
| 模型 | LLaMA-2-7B (`/root/llama2-7b/`) |
| 序列长度 | 4096 tokens |
| 校准样本数 | 16 |
| 校准数据 | PG-19 test split (预分词) |
| 分析层 | 0, 4, 8, 16, 31 |
| 精度 | float16 |
| head_dim | 128 (64 pairs) |
| num_heads | 32 (KV heads) |
| RoPE base | 10000.0 |
| 稀疏性能量目标 | 95% |
| 稀疏性采样 | 256 traces/layer |
| 小波族 | Daubechies-4 (db4) |

### 1.3 方法

1. 将 LLaMA-2-7B 加载至 GPU，在 `k_proj` 上注册 forward hook 捕获 **pre-RoPE** Key 状态。
2. 对 pre-RoPE K 手动施加 RoPE 旋转得到 **post-RoPE** K。
3. 将每对隐维视为复数序列 c_i(t) = k_{2i}(t) + j·k_{2i+1}(t)，沿序列轴做 DFT，计算功率谱 |C_i[ω]|²，在 batch 和 heads 上取平均。
4. 对 pre-RoPE K 采样序列轴 trace，分别计算 DCT-II、DFT (rfft)、DWT (db4) 三种基底下达到 95% 能量所需的最小系数占比。

## 2. 频谱分析结果

### 2.1 Pre-RoPE 频谱特征

在所有 5 层中，pre-RoPE K 的频谱都呈现一致的特征：

- **能量高度集中于低频**（bin 0 附近），衰减速度快，符合 AR(1) 类平稳信号的特性。
- 不同 pair 之间的频谱形状相似，仅幅度不同。
- 这验证了 FreqKV 论文的核心观察：pre-RoPE K 沿序列轴近似低通信号，DCT-II 低通截断是合理的压缩策略。

### 2.2 Post-RoPE 频谱特征

施加 RoPE 后，频谱分布发生了显著变化，与 DFT 调制定理的预测高度吻合：

- **Pair 0 (θ₀ = 1.0 rad/sample)**：能量从零频移开，在 bin ~652 (= round(1.0 × 4096 / 2π)) 附近出现清晰峰值。该峰值位置与调制定理预测的 n₀ 精确对齐。

- **Pair 32 (θ₃₂ ≈ 0.01 rad/sample)**：频移量小得多，峰值仍集中在低频区域，但相对 pre-RoPE 有可观测的右移。

- **Pair 63 (θ₆₃ ≈ 0.0001 rad/sample)**：频移几乎为零，post-RoPE 频谱与 pre-RoPE 几乎无差异。

这一规律在所有层（Layer 0, 4, 8, 16, 31）中一致出现，中深层（Layer 16, 31）的效应尤其清晰，因为 K 的语义内容更丰富、信噪比更高。

### 2.3 调制定理验证结论

**DFT 调制定理在 LLaMA-2-7B 的真实 Key 状态上经验性成立。**

关键推论：FreqKV 的 DCT-II 低通保留 bin [0, L) 的策略对于高索引 pair（小 θ，能量仍在零频）是有效的，但对低索引 pair（大 θ，能量已被 RoPE 搬移到高频段）会**系统性丢失信号**。这些低索引 pair 恰好编码短程位置细节（高频旋转），这可能解释了 FreqKV 在 Needle-in-a-Haystack 和代码检索等依赖精确位置的任务上的性能下降。

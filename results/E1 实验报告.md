# E1 实验报告：Post-RoPE 频率梳验证（主张 C1）

## 1. 实验目的

验证主张 **C1**：post-RoPE 的 K 沿序列轴存在**确定性频率梳**——第 i 对 pair 的
DFT 能量峰精确出现在 n_i = round(θ_i · N / 2π)，其中 θ_i = base^(-2i/d)。

这是 RST-KV 整条技术路线的理论基础。如果梳齿位置与 θ_i 不匹配，则
"RoPE 匹配带通"失去理论根据。

## 2. 实验配置

| 参数 | 值 |
|------|-----|
| 模型 | LLaMA-2-7B (`/root/llama2-7b/`) |
| RoPE base | 10000.0 |
| 序列长度 N | 4096 tokens |
| 校准样本数 | 8 |
| 校准数据 | PG-19 test split（预分词） |
| 分析层 | 0, 8, 16, 31 |
| 分析 pair | 0, 1, 4, 8, 16, 32, 63 |
| head_dim | 128（64 pairs）|
| num_heads | 32 |
| 精度 | float16 |

## 3. 方法

1. 加载 LLaMA-2-7B，在每层 `k_proj` 上注册 forward hook 捕获 **pre-RoPE** K。
2. 对 pre-RoPE K 手动施加 RoPE 得到 **post-RoPE** K。
3. 将每对隐维视为复数序列 c_i(t) = k_{2i}(t) + j·k_{2i+1}(t)，沿序列轴做 DFT。
4. 计算功率谱 |C_i[ω]|²，在 batch 和 heads 上取平均。
5. 对每个 pair 定位实测峰位 (observed peak)，与理论预测 n_i 对比。

### 4.2 汇总统计

- **Pair 0–32（共 24 个测量点）**：`|peak-pred|` = 0（全部精确命中），无一例外。
- **Pair 63（4 个测量点）**：`|peak-pred|` = 1，θ₆₃ ≈ 1.15×10⁻⁴，预测 n₆₃ = 0，
  实测峰在 bin 1。由于 θ 极小，频移几乎为零，bin 0 和 bin 1 之差在 DFT 分辨率范围内。
- **Sharpness（梳齿锐度）**：Pair 0–32 的 sharpness 范围为 271–10850，均远大于 1，
  表明梳齿极其尖锐——post-RoPE 能量高度集中在预测的齿位上。
- **DC fraction**：Pair 0–32 的 DC frac 均 < 0.003，说明能量确实被 RoPE 搬离了
  零频。Pair 63 的 DC frac 达 0.18–0.66（层越深越高），说明高索引 pair 的能量
  仍留在 DC 附近，与理论一致。
